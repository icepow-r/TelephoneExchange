# Telephone Exchange System (Мини-АТС)

Клиент-серверная система симуляции телефонной мини-АТС на C# с WPF интерфейсом.

## Параметры системы
- Количество абонентов: 7 (2N-1, где N=4)
- Лимит соединений АТС: 4 одновременных соединения (2K, где K=2)
- Длина телефонного номера: 3 цифры

## Структура проекта
- `TelephoneExchange.Server` - серверное консольное приложение
- `TelephoneExchange.Client` - клиентское WPF приложение

## Технологии
- C# .NET
- WPF
- TCP Sockets

## Запуск
1. Запустите TelephoneExchange.Server
2. Запустите несколько экземпляров TelephoneExchange.Client
3. Подключитесь к серверу (по умолчанию 127.0.0.1:5000)
